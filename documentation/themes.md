# Темы оформления

## Встроенные темы

mkpy поставляется с двумя встроенными темами:

### light

Светлая тема с белым фоном.

```python
docs = Docs(theme="light")
```

### dark

Темная тема с темным фоном.

```python
docs = Docs(theme="dark")
```

## Что включают темы

Встроенные темы включают:

- Типографика (шрифты, размеры, межстрочный интервал)
- Стили для заголовков (h1-h6)
- Стили для кода (inline и блоки)
- Стили для таблиц
- Стили для цитат (blockquote)
- Стили для изображений
- Стили для списков
- Адаптивность для мобильных устройств

## Собственная тема

Вы можете переопределить стили с помощью параметра `custom_css`:

```python
docs = Docs(
    theme="dark",
    custom_css="""
    body {
        font-size: 18px;
    }
    h1 {
        color: #ff6600;
    }
    """
)
```
